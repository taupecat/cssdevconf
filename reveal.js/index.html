Media Query Bubbling:
Jedi Master

SCSS:


@mixin bp($breakpoint) {
  @if $breakpoint == medium {
    @media all and (min-width: (690px / 16px * 1em)) { @content; }
  }
}

.sidebar {
  background-color: #e5e5e5;
  font-size: 1.2em;

  @include bp(medium) {
    float: left;
    width: percentage(400px / 1280px);
  }
}

Credit: css-tricks.com

@extend + %placeholder-selectors

Lets you aggregate your classes around a set of common properties.


%clearfix {
  &:before,
  &:after {
    content: "";
    display: table;
  }

  &:after {
    clear: both;
  }
}

.footer {
  @extend %clearfix;
}


.footer:before,
.footer:after {
  content: "";
  display: table;
}

.footer:after {
  clear: both;
}

Doesn't work in @media queries.
